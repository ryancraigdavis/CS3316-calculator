# Simple Calculator

This program takes a few different inputs and can spit out various results from mathematical formulas.
Your task is to create simple unit tests for each of these formulas.

1. Setup your virtual environment
2. Install Pytest as a dependency
3. Add your testing file and folder structure
4. Set your PYTHONPATH variable to the top level
5. Write your tests
6. Run pytest by running the command "pytest -v" - -v for verbose
7. Submit a screenshot of your passed test
